LOGOS DEL ERP

  icono.png
    → Favicon (pestaña del navegador), apple-touch-icon y base para los PNG
      public/icons/icon-192.png / icon-512.png al hacer npm run build (icono al instalar la app).
      Sustituye este archivo cuando mejores la calidad del icono; luego vuelve a hacer build.

  logo-light.png
    → Menú lateral y pantalla de login (app/erp-branding.config.json → logoLightPath).

  logo-light-large.png (opcional)
    → Respaldo para generar iconos PWA si no existe icono.png ni logo-light.png.

  logo.png
    → Logo principal (cabeceras, right-sidebar, etc.). Respaldo final para iconos PWA.

  public/icons/icon-192.png y icon-512.png
    → Se regeneran en cada npm run build; no hace falta editarlos a mano.
